
# Expense Tracker

A modern and responsive Expense Tracker Web Application built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Add Income and Expenses
- Real-Time Balance Calculation
- Transaction History
- Delete Transactions
- Dark / Light Mode
- Data Persistence using localStorage
- Responsive User Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📚 Learning Outcome

This project helped me understand:

- DOM Manipulation
- Event Handling
- CRUD Operations
- Arrays and Objects
- localStorage
- Dynamic UI Rendering
- Responsive Web Design

---

## 💡 How It Works

Users can add income or expense transactions with an amount and description.  
The application automatically calculates:

- Total Income
- Total Expense
- Current Balance

All transaction data is stored in browser localStorage, so data remains saved even after refreshing the page.
